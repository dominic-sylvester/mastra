# Audience

Mastra's primary audience is TypeScript developers building AI-powered applications, from early prototypes to production systems. Secondary audiences include engineering leaders evaluating agent frameworks and technical content creators covering the AI ecosystem.

## Primary audience

- **TypeScript developers**: Building agents, workflows, and tools in production codebases.
- **Full-stack engineers**: Adding AI capabilities to Next.js, React, Node.js, or similar stacks.
- **Platform teams**: Embedding agent functionality into existing products.

## How to write for this audience

- Lead with concrete capabilities and working code.
- Avoid generic AI jargon. Be specific about what Mastra does.
- Assume familiarity with TypeScript and modern web frameworks.
- Link to runnable examples rather than abstract descriptions.
